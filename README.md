## What is supaBase ?
supabase is open Source firebase alternative built on the top of postgress.

## What is Firebase ?
Firebase is supabase these all are backend as service baas.

## What is backend as service ?
They basically gives you everything like auth, db, stograge, realtime, crud Api

## Features of SupaBase:-
1. Database-postgressSql
2. Auth- Email, oAuth, magic Links, more
3. Storage
4. Edge Function
5. RealTime:- for live updates without manual poling


## Why developer Like Supabase
1. instant backend setup
2. open Source & sql based
3. auto generated apis
4. realtime
